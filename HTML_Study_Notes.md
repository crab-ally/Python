# HTML

## 1. 웹과 HTML

### 1.1 웹
- **인터넷**: 전 세계 컴퓨터가 연결된 거대한 네트워크
- **웹(World Wide Web)**: 인터넷 위에서 동작하는 서비스
    - **동작 원리**
        - **요청(Request)**: 브라우저가 서버에 웹 페이지를 요청
        - **처리**: 서버가 요청을 처리하고 웹 페이지를 준비
        - **응답(Response)**: 서버가 파일들(HTML, CSS, JS 등)을 브라우저에 전송
        - **렌더링(Rendering)**: 브라우저가 받은 파일들을 해석해서 웹 페이지로 변환

### 1.2 HTML
- **HTML(HyperText Markup Language)**: 웹페이지의 구조와 내용을 정의하는 마크업 언어
    - **마크업 언어**: 컴퓨터에게 '이 부분은 제목이야', '이 부분은 문단이야'라고 알려주는 표시 언어
    - **특징**
        - **구조화된 문서**
        - **태그 기반**: <태그명> </태그명> 구조
        - **하이퍼링크**: 다른 웹 페이지로 연결하는 링크 생성 가능
        - **플랫폼 독립적**: 어떤 기기에서든 동일하게 표시

### 1.3 HTML 문서의 기본 구조
- **<!DOCTYPE html>**: HTML 문서임을 선언
- **<html>**: 모든 내용을 감싸는 최상위 요소
- **<head>**: 사용자에게는 보이지 않지만 중요한 정보들이 들어가는 곳
    - **<title>**: 웹 브라우저 탭에 표시되는 제목
- **<body>**: 사용자가 실제로 보게 될 내용들이 들어가는 곳

### 1.4 태그와 속성의 기본 구조
- **태그 기본 구조**: <태그명 속성="값"> 내용 </태그명>
    - 여는 태그: <태그명>
    - 닫는 태그: </태그명>
    - 속성: 태그에 추가 정보 제공
    - 내용: 태그 사이에 들어가는 실제 콘텐츠

### 1.5 개발자 도구
- **개발자 도구로 할 수 있는 일들**
    - HTML 구조 확인: 웹페이지가 어떤 태그들로 구성되어 있는지
    - CSS 스타일 분석: 각 요소에 어떤 디자인이 적용되었는지
    - 실시간 수정: 코드를 바로 수정해서 결과를 즉시 확인
    - 오류 찾기: JavaScript 에러나 문제점 발견
    - 성능 분석: 페이지 로딩 속도나 최적화 상태 점검
- **개발자 도구 단축키**
    - Windows/Linux: F12
    - Mac: Cmd + Option + I
    - 마우스 우클릭: "검사" 또는 "요소 검사" 선택

## 2. HTML 기본 태그

### 2.1 제목
- **<h1> ~ <h6>**: 제목 태그 (숫자가 작을수록 더 크게 표시됨)

### 2.2 문단과 줄 바꿈
- **<p> [paragraph]**: 문단(텍스트 블록)
- **<br> [break]**: 줄 바꿈

### 2.3 목록
- **<ul> [unordered list]**: 순서 없는 목록 (불릿 포인트)
- **<ol> [ordered list]**: 순서 있는 목록 (번호)
- **<li> [list item]**: 목록 항목
    - <ul>이나 <ol> 안에서만 사용할 수 있다

### 2.4 링크
- **<a> [anchor]**: 링크
    - **속성**
        - **href="연결할 주소"**
            - 외부링크
                - https:// 로 시작하는 완전한 URL
            - 내부링크: 같은 사이트 내 다른 페이지로 이동
                - 파일명
            - 앵커링크: 같은 페이지 내 특정 위치로 이동
                - 이동할 위치의 태그에 id="목표" 속성 추가
                - 링크에 href="#목표"

### 2.5 이미지
- **<img> [image]**: 이미지
    - **속성**
        - **src="이미지 주소"**
        - **alt="이미지 설명"**: 이미지를 볼 수 없을 때 표시되는 대체 텍스트
        - **width="값"**: 가로 크기 (단위: 픽셀)
        - **height="값"**: 세로 크기 (단위: 픽셀)

### 2.6 강조와 서식
- **<strong>**: 굵은 글씨 (중요한 내용)
- **<em> [emphasis]**: 기울임 글씨 (강조하는 내용)

### 2.7 테이블
- **외부 뼈대**
    - **<table>**: 전체 테이블을 감싸는 컨테이너
    - **<thead>**: 테이블의 머리글 영역
    - **<tbody>**: 테이블의 본문 영역
- **내부**
    - **<tr> [table row]**: 행
    - **<th> [table header]**: 제목 셀 (굵게 표시되고 가운데 정렬)
    - **<td> [table data]**: 일반 데이터 셀
- **속성**
    - **colspan="값"**: 열 병합
    - **rowspan="값"**: 행 병합

### 2.8 인용문
- **<blockquote>**: 긴 인용문을 독립된 블록으로 표시
- **<q>**: 짧은 인용문을 문장 안에 표시
- **<cite>**: 인용 출처 표시
- **<code>**: 짧은 코드를 문장 안에 표시
- **<pre>**: 긴 코드 블록을 원본 형태로 표시 (들여쓰기, 줄 바꿈 등)

### 2.9 구분선과 레이아웃
- **<hr> [horizontal rule]**: 수평선
- **<div> [division]**: 블록 레벨 요소 (큰 영역을 나누는 컨테이너 역할)
- **<span>**: 인라인 요소 (문장 속에서 특정 부분만 스타일을 다르게 할 때)

## 3. HTML 폼과 입력 요소

### 3.1 폼
- **<form>**: 웹페이지에서 사용자가 정보를 입력하고 서버로 전송할 수 있게 해주는 기능
    - **속성**
        - **action="연결할 주소"**: 폼 데이터를 전송할 서버의 주소
        - **method="전송 방식"**: 폼 데이터를 전송할 방식
            - **get**: 폼 데이터를 URL 뒤에 붙여서 전송 (보안 취약)
            - **post**: 폼 데이터를 서버에 직접 전송

- **<button>**: 클릭 가능한 버튼
    - **속성**
        - **type="submit"**: 폼 제출 버튼 [기본값]
        - **type="reset"**: 폼 초기화 버튼
        - **type="button"**: 일반 버튼
        - **disabled**: 비활성화

### 3.2 텍스트 입력
- **<input>**: 사용자 입력을 받는 필드
    - **속성**
        - **데이터 종류**: 어떤 종류의 입력을 받을지 결정
            - **type="text**: 일반 텍스트
            - **type="password"**: 비밀번호
                - 사용자가 입력하는 글자가 화면에는 점(●)이나 별표(*)로 표시
            - **type="email"**: 이메일
            - **type="number**: 숫자
            - **type="hidden"**: 사용자에게 보이지 않지만 폼이 전송될 때 함께 서버로 전송
        - **설정**
            - **size="값"**: 입력창의 크기 조절
            - **maxlength="값"**: 최대 입력 글자 수 제한
            - **minlength="값"**: 최소 입력 글자 수 제한
            - **readonly**: 읽기 전용 (수정 불가)
            - **placeholder="안내 문구"**: 입력창 안에 보이는 회색 안내 문구
            - **required**: 필수 입력 항목으로 지정
        - **전송 데이터**
            - **name="값"**: 폼 데이터를 서버로 전송할 때 사용할 이름
            - **value="값"**: 화면에 표시되는 기본값 + 서버로 보내지는 기본값

### 3.3 체크박스와 라디오 버튼

#### 3.3.1 체크박스
- **<input type="checkbox">**
- **특징**
    - 다중 선택
    - 각각 독립적으로 체크/해제
    - 각각 다른 name 속성 사용 (서버에 다르게 전달됨)

#### 3.3.2 라디오 버튼
- **<input type="radio">**
- **특징**
    - 단일 선택
    - 하나를 선택하면 다른 것은 해제됨
    - 같은 name 속성으로 그룹핑

### 3.4 드롭다운
- **<select>**: 드롭다운 상자 전체
    - **multiple**: 다중 선택 가능
        - Windows: Ctrl + 클릭
        - Mac: Command + 클릭
    - **size="값"**: 화면에 보이는 항목의 수
    - **disabled**: 비활성화

- **<option>**: 드롭다운 각각의 선택 항목
    - **value="값"**: 서버로 전송될 값
    - **selected**: 기본적으로 선택된 항목

- **<optgroup>**: 옵션 그룹화
    - <option>을 묶는 역할
    - **label="옵션 그룹명"**: 옵션 그룹명 표시

### 3.5 긴 글 입력
- **<textarea>**: 여러 줄 텍스트 입력
    - **속성**
        - **rows="값"**: 세로 크기 (기본값: 2)
        - **cols="값"**: 가로 크기 (기본값: 20)
        - **placeholder="안내 문구"**: 입력창 안에 보이는 회색 안내 문구

### 3.6 라벨
- **<label>**: 입력 요소 설명(라벨)
    - **장점**
        - 접근성 향상: 스크린 리더와 같은 보조 기술이 입력 필드의 목적을 정확히 파악할 수 있음
        - 사용 편의성: 라벨을 클릭하면 해당 입력창에 포커스가 감
    - **label과 input 연결**
        - **for="입력요소id"**: 라벨 클릭하면 입력창에 포커스 감
        - **label로 input 감싸기**
