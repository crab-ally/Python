# Javascript

## 1. 기초

### 1-1. 기본

| 명령어 | 내용 |
| --- | --- |
| `//`, `/* */` | 주석 |
| `;` | 문장의 끝을 알림 |
| `+`, `,` | 문자열 더하기 |
| `<br>` | 줄바꿈 |
| `&nbsp` | 공백 |
| `typeof` | 자료형 확인 |

> 자료형 : String, Number(정수, 실수), Boolean ...

### 1-2. 화면출력

| | `document.write()` | `console.log()` |
|---|---|---|
| 사용이유 | 화면(웹브라우저)에 출력 | 개발자 도구(F12)에 출력 |
| 줄바꿈 | `<br>` | 자동 줄바꿈 |
| 공백 | `&nbsp` | 여러 개의 공백 유지 |

### 1-3. 변수 선언

```javascript
let a = 10;
let b;
b = "Hello";
```

### 1-4 변수명명 규칙

- 대소문자 구분
- 공백 불가
- 숫자로 시작 불가
- 특수문자는 `_`, `$` 만 가능
- 예약어(키워드) 사용불가
