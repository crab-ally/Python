# Javascript

## 1. 기초

### 1-1. 기본

| 연산자/명령어 | 내용 |
| --- | --- |
| `//`, `/* */` | 주석 |
| `;` | 문장의 끝을 알림 |
| `+`, `,` | 문자열 더하기 |
| `<br>` | 줄바꿈 |
| `&nbsp` | 공백 |
| `typeof` | 자료형 확인 |

> 자료형 : String, **Number(정수, 실수)**, Boolean ...

### 1-2. 화면출력

| | `document.write()` | `console.log()` |
|---|---|---|
| 사용이유 | 화면(웹브라우저)에 출력 | 개발자 도구(F12)에 출력 |
| 줄바꿈 | `<br>` | 자동 줄바꿈 |
| 공백 | `&nbsp` | 여러 개의 공백 유지 |

### 1-3. 변수 선언

```javascript
let a = 10;
let b;
b = "Hello";
```

### 1-4 변수명명 규칙

- 대소문자 구분
- 공백 불가
- 숫자로 시작 불가
- 특수문자는 `_`, `$` 만 가능
- 예약어(키워드) 사용불가

## 2. 산술연산자

| 연산자/명령어 | 내용 | 특이사항 |
| --- | --- | ---|
| `+` | 덧셈 | |
| `-` | 뺄셈 | |
| `*` | 곱셈 | |
| `/` | 나누기 | 0으로 나누면 `Infinity` 반환 | | 
| `parseInt(a/b)` | 나눗셈 몫 구하기 | 반환값 타입 - Number |
| `%` | 나머지 | 0으로 나누면 `NaN` 반환 |

### 2-1. 할당 연산자

| 연산자 | 내용 |
| --- | --- |
| `+=` | 증가 할당 |
| `-=` | 감소 할당 |
| `*=` | 곱셈 할당 |
| `/=` | 나눗셈 할당 |
| `%=` | 나머지 할당 |

### 2-2. 증감 연산자

| 연산자 | 내용 | 특이사항 |
| --- | --- | ---|
| `++a` | 전위 증가 | |
| `a++` | 후위 증가 | 우선순위 마지막(출력 이후 증가 처리) |
| `--a` | 전위 감소 | |
| `a--` | 후위 감소 | 우선순위 마지막(출력 이후 감소 처리) |

### 2-3. 소수점 처리

소수 부분이 0이면 생략

| 명령어 | 내용 | 특이사항 |
| --- | --- | ---|
| `실수.toFixed(자릿수)` | 소수점 자릿수 지정 | 반올림 처리, 남은 자릿수를 0으로 채움, 반환값 타입 - String |

### 2-4. 랜덤

| 연산자 | 내용 |
| --- | --- |
| `Math.random()` | 0.0 이상 1.0 미만의 실수를 반환 |
| `Math.floor()` | 소수점 버림 |
